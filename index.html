📲 Установить приложение

Установи IziSerb прямо на экран телефона — работает как полноценное приложение.

🍎 iPhone / iPad (Safari)
  1. Откройте cabinet.iziserb.com в Safari
  2. Нажмите кнопку «Поделиться» ⬆
  3. Выберите «На экран "Домой"»
  4. Нажмите «Добавить»
🤖 Android (Chrome)
  1. Откройте cabinet.iziserb.com в Chrome
  2. Нажмите три точки ⋮
  3. Выберите «Добавить на главный экран»
  4. Нажмите «Установить»
